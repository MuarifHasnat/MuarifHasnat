### 👨‍💻 My Background & Focus

I am currently in **Level 3, Term 1** of my B.Sc. in Computer Science Engineering at **BAUST, Saidpur**. My development philosophy revolves around clean coding principles and bridging the gap between high-level user interfaces and low-level hardware interactions.

**🎯 Current Academic & Project Focus:**
- 📱 **Software Engineering:** Developing an **Equipment Borrowing System** for Android using Kotlin, following the Incremental Model architecture.
- ⚙️ **Low-Level Programming:** Writing and debugging **8086 Assembly Language** (working with MS-DOS interrupts, memory registers, and data segments).
- 🧠 **Theoretical CS:** Implementing algorithms for **Compiler Design**, specifically analyzing Context-Free Grammars and left recursion elimination.
- 🌐 **Networking:** Deep diving into the **OSI Model** and core computer networking protocols.
